# ComparEdge API Documentation

REST API for [ComparEdge](https://comparedge.com) — structured data for 331 SaaS products.

## Endpoints

- [Products](https://comparedge-api.up.railway.app/docs)
- [Categories](https://comparedge-api.up.railway.app/docs)
- [Comparisons](https://comparedge-api.up.railway.app/docs)
- [Download Endpoints](download-endpoints.md)

## Quick Start

```bash
# Get all products
curl https://comparedge-api.up.railway.app/api/v1/products

# Get a specific category
curl https://comparedge-api.up.railway.app/api/v1/categories

# Download CSV
curl -O https://comparedge-api.up.railway.app/download/csv
```
